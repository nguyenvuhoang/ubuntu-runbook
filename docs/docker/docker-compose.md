# Docker Compose Commands

## Start services

```bash
docker compose up -d
```

## Stop services

```bash
docker compose down
```

## Recreate containers

```bash
docker compose up -d --force-recreate
```

## Pull latest images

```bash
docker compose pull
```

## Validate compose file

```bash
docker compose config
```

## View containers

```bash
docker ps
```

## Tags

`docker`, `docker-compose`, `compose`, `container`
